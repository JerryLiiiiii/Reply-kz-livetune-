# GUIDE
To use this project, copy the code [**reply-mimikoppi_REPL.strudel.js**](src/reply-mimikoppi_REPL.strudel.js) into the editor at [strudel.cc](https://strudel.cc).

Or if you prefer to [use strudel in your IDE](https://strudel.cc/learn/faq/#can-i-use-strudel-with-my-ide), use [**reply-mimikoppi_REPL_noViz.strudel.js**](src/reply-mimikoppi_REPL_noViz.strudel.js) instead, as IDEs have limited support for strudel's visualization features.

## Important
The `BassDrum` sounds different in my VS Code (e.g., possibly odd). Please **open an issue** if you encounter the same problem.I primarily edit in the strudel REPL, and simply copy the code to the `.strudel.js` files, and comment out all the visualization functions to run tests in VS Code.
